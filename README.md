## 🏥 Health Data Analysis Dashboard

This project presents an interactive health data dashboard created using Excel and Power BI.

### 📊 Tools Used

* Microsoft Excel
* Microsoft Power BI

### 📁 Dataset

The dataset contains health-related information such as patient records, disease types, and trends.

### 📈 Dashboard Features

* Patient data analysis
* Disease distribution
* Trend visualization
* Interactive filters
